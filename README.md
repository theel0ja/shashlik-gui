# shashlik-gui

Graphical user interface for [Shashlik](http://www.shashlik.io/).

## Depencies

* zenity
* [shashlik](http://www.shashlik.io/download/)

## Installation

1. Install zenity
2. Install shashlik-gui:
```
git clone https://github.com/theel0ja/shashlik-gui.git
cd shashlik-gui
chmod +x install-app.sh
./install-app.sh
```
